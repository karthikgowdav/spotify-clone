# spotify-clone-web-🎵
Responsive Spotify Clone web application built to replicate the core features and UI of Spotify. This project focuses on front-end design, music playback functionality, and user interaction, providling a real-world experience of building a modern music streaming interface



## 🚀 Features
- Play / Pause functionality
- Song progress bar
- Next / Previous buttons
- Responsive UI

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
